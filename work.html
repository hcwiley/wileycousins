flamejets

A mechatronics project that Michael Cousins built with Chris Yarka. This box shoots four propane powered flames into the air. A PIC32 microcontroller varies the heights.






rgb lighting module

A daisy-chainable lighting module to brighten up your existence (and/or living space). These modules receive power and communicate over an ehternet cable, but don't plug it into your router as it's likely to blow it up, hence the NOT ETHERNET. Check it out on Github

kinobi

Chapman Snowden (CEO), Phil Winfield (Designer), Zack Dever (Developer), and Cole Wiley (Developer) worker together for 6 months to launch Kinobi; a web app that allowed users to receive realtime feedback while learning kinesthetic skills. Kinobi presented at Tech Crunch Disrupt NYC 2012: Startup Alley & Tech Crunch Disrupt SF 2012: Startup Battlefield

riverless walk

"riverless walk" is an installation that bridges the gap of presenting three dimensional objects on a two dimensional surface. Cole Wiley wrote custom software (writen in a mixture of Java and Arduino using OpenNI for working with the Kinect) that tracks bodies, and communicates with an Arduino to control a motor. The motor rotates a custom designed platform so that the projector and kinect follow the user around the room. The screen encompasses a half circle with a 16ft radius, making it a 50ft long screen.







a community gallery

A Community Gallery is both a website and a site specific installation. The website is running on Django (Python), and the installation is running a combination of Java, Node.js, and OpenNI. The installation has been majorly refactored, as it was initially written in C++ in VisualStudios.
